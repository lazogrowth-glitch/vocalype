# Vocalype Product Knowledge

Vocalype is a local-first voice dictation SaaS.

Promise:

Speak -> text appears -> paste anywhere.

Main benefits:

- Offline by default
- Fast dictation
- Private
- Simple UX
- Works across apps
- Useful for writing, coding, emails, notes, ChatGPT prompts, professional documents

Core user journey:

1. User lands on website
2. User understands product in less than 5 seconds
3. User downloads app
4. User logs in or activates license
5. User grants permissions
6. User chooses model/settings
7. User starts dictating
8. Text appears
9. User pastes text into another app
10. User understands value and upgrades

Most important activation event:

First successful dictation.

North Star:

Number of successful dictations per active user per week.

Operational priorities:

- Remove anything blocking first successful dictation.
- Make privacy and offline behavior obvious before trust is lost.
- Improve model quality with repeatable benchmarks, not guesses.
- Convert interested users by showing the product working in real apps.
- Prefer small measured improvements over large speculative rewrites.
