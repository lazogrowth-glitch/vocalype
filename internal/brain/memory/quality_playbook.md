# Vocalype Quality Playbook

Core product quality priorities:

1. First successful dictation
2. Low dictation latency
3. Low memory usage
4. Stable activation/license flow
5. No crashes
6. Clear permissions
7. Accurate transcription
8. Good model presets
9. Offline reliability
10. Clear error messages

Every quality improvement must include:

- metric
- baseline
- target
- test method
- risk
- expected impact
- rollback plan

Never accept vague improvements.

No quality claim is valid without measurement.
