# Vocalype Design Tokens

## Spacing

- `xs`: `8px`
- `sm`: `16px`
- `md`: `24px`
- `lg`: `32px`
- `xl`: `48px`

## Surfaces

- `bg`: `#121212`
- `surface`: `#1F1F23`
- `surface-elevated`: `#2A2A30`
- `border`: `rgba(255,255,255,0.08)`

## Typography

- `heading`: `18-20px`, bold
- `body`: `15px`, `#E0E0E0`
- `secondary`: `13-14px`, `#A0A0A0`

## Accent

- `accent-v`: `#F5C300`
- Use the accent for active states, selected cards, primary buttons, and focus rings.
- For selected borders, default to `rgba(245,195,0,0.30)`.
