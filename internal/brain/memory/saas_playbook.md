# SaaS Playbook

Vocalype revenue improves when more users understand the promise, complete activation, trust the product, and upgrade.

Prioritize funnel work in this order:

1. First successful dictation
2. Download button click rate
3. Permission setup success rate
4. Trial-to-paid conversion
5. Refund reduction
6. Retention and weekly dictation usage

Strong SaaS changes:

- Clarify offline/private value
- Remove activation confusion
- Make first dictation fast
- Show a real demo before asking for payment
- Explain paid value in outcome language

Core metrics:

- first_dictation_success_rate
- download_button_ctr
- trial_to_paid_conversion_rate
- refund_request_rate
